B) CSS Files and Structure - top

I'm using a main CSS file in this theme called styles.css. It holds all the styles implemented by the theme. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values could be overridden somewhere else in the file.

The styles.css file contains all of the specific stylings for the page. The file is separated into sections using this table of contents:

Advanced users can install the free plugin Jetpack and activate its option to enter your own CSS rules.

If you would like to edit a specific section of the site, simply find the appropriate section from the table of contents in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more specificity.

I.E.

	#wrap #primary a {
		color: #someColor;
	}

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

The SASS files have also been included in the sass folder.

C) JavaScript - top

This theme at the current settings imports three Javascript files through custom-functions.php. Those javascript files are in the "js" folder.

  1. jQuery
  2. Modernizr
  3. Navigation Script
  4. REM Unit Polyfill
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Modernizr is a collection of superfast tests which run as your web page loads, then the results are used to tailor the best experience to the user.
  3. The navigation script is a small file to make the expanded navigtion menu more friendly on touch devices.
  4. REM Unit Polyfill is a small file that tests for REM unit support. If the browser does not support it, the script offers a quick fix so that the user experience is still beautiful.

There are other javascript files in the "js" folder that may be loaded in for extra convinience such a "skip-link-focus-fix.js".

D) Customize Theme - top

You can customize the appearance of the theme to fit your needs. Access these options at “Appearance” → “Customize”. A preview window will open up with a new sidbar on the left. Here you’ll find all available options.

To use other Google Fonts you can download and install the free Easy Google Fonts plugin: https://wordpress.org/plugins/easy-google-fonts/

After that you can chose the font that fits your needs. Be aware that some fonts render bigger than others, so check the theme after switching fonts for glitches.

Font Awesome has also been included. For reference on how to use the icon you want, visit http://fontawesome.io/icons/.

E) Translations - top

You can create other languages, too. The free app PO-Editor will help you translating the theme into your language. Detailed instructions can be found here: http://code.tutsplus.com/tutorials/translating-your-theme--wp-25014 As an alternative you can use the Wordpress plugin Loco to translate the theme right inside the admin area: https://wordpress.org/plugins/loco-translate/.
